# Car Price Prediction using CarDekho Dataset

## Overview
This project predicts car prices using machine learning models based on features like mileage, fuel type, and vehicle age.

## Steps Performed
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Selection
- Model Building (Linear Regression, Decision Tree, Random Forest)
- Model Evaluation using R² Score

## Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Result
Random Forest performed best among the models with highest accuracy.
